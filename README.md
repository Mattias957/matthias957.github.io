# matthias957.github.io

Mitt digitala CV / Hemsida

This is a [Next.js](https://nextjs.org) project for my personal portfolio website.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3001](http://localhost:3001) with your browser to see the result.

## Deployment

This project is automatically deployed to GitHub Pages via GitHub Actions. The site is available at [https://matthias957.github.io](https://matthias957.github.io).
